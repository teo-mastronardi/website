Contact Resource Management Application

As noted on my resume, I spent a work term creating a Contact Resource Management Application with the company Tech-it-easy During this work term, It was my responsibility to create this web application from scratch while using the customers template as a reference.



Inputting Database

In the application, users can create a new profile for new customers by inputting information about them such as: Names, addresses, date of births and insurance information.


Searching Database

This information was then stored in our database using SQL where we can then search in real-time for the customers by typing for any valuesvinputted in the database corresponding to that customer. This was done by using Ajax requests.


Inputting Files

In the application you are also able to input documents including PDF files to the customers profile.
